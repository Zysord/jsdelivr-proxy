CDN 代理服务

管理面板

npm 包

访问 npm 包资源，支持版本控制和文件路径访问。

GitHub 仓库

访问 GitHub 仓库文件，支持分支、标签和具体文件路径。

WordPress

访问 WordPress 插件和主题资源。

使用方法

npm 包

https://your-domain/npm/package-name@version/file

示例：/npm/jquery@3.6.0/dist/jquery.min.js

GitHub 仓库

https://your-domain/gh/user/repo@version/file

示例：/gh/jquery/jquery@3.6.0/dist/jquery.min.js

WordPress

https://your-domain/wp/plugins/plugin-name/file
https://your-domain/wp/themes/theme-name/file

示例：/wp/plugins/woocommerce/assets/js/frontend/cart.min.js

注意事项

  • 所有请求的资源必须在白名单中才能访问
  • 支持版本号和具体文件路径访问
  • 缓存时间默认为 24 小时
  • 如需添加白名单，请联系管理员
